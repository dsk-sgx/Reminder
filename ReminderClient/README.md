# Reminder
日々の業務や学習したことの備忘録を残しておくクライアントアプリケーションです。

## Version
1.0.0

## Description
electronでWindowを表示し、データはIndexedDatabaseに保存します。  
サーバとの同期を実装予定です。

## Install
    git clone https://github.com/sugiyamads/Reminder.git
    cd Reminder/ReminderClient/
    npm install
    
## Usage
    npm start
    
＊ プラットフォームごとにパッケージ化する予定です。
